# StreamLine SaaS — Product Documentation

## Getting Started
1. Sign up at app.streamline-saas.com
2. Create your first workspace
3. Invite team members via email
4. Create your first project and add tasks

## Features & How-To

### Task Management
- **Create a task**: Click "+" in any project → fill title, description, assignee, due date
- **Kanban board**: Drag tasks between columns (To Do, In Progress, Review, Done)
- **Gantt chart**: Toggle to Gantt view for timeline visualization
- **Subtasks**: Click a task → Add subtask for breaking down work
- **Labels**: Color-coded labels for categorization (Bug, Feature, Urgent, etc.)

### Team Collaboration
- **Chat**: Real-time messaging in project channels or direct messages
- **Video calls**: Click camera icon in any chat to start a call (up to 25 participants)
- **@mentions**: Tag team members in tasks or comments for notifications
- **Activity feed**: See all recent updates across your workspace

### File Sharing
- **Upload**: Drag and drop files into any task or chat (max 100MB per file)
- **Supported formats**: PDF, DOCX, XLSX, PNG, JPG, MP4, ZIP
- **Version history**: All files maintain version history (last 30 versions)
- **Sharing**: Generate public links or restrict to workspace members

### Time Tracking
- **Start timer**: Click the clock icon on any task to begin tracking
- **Manual entry**: Add time entries retroactively
- **Reports**: Export time reports by project, user, or date range (CSV/PDF)
- **Billing integration**: Time tracked can be used for client invoicing

### Integrations
- **Slack**: Two-way sync — get StreamLine notifications in Slack channels
- **GitHub**: Link pull requests to tasks, auto-update task status on merge
- **Jira**: Import existing Jira tickets into StreamLine projects
- **Google Workspace**: SSO login, Google Drive file linking
- **API**: RESTful API available on Professional and Enterprise plans

### Account Management
- **Password reset**: Settings → Security → Change Password (or "Forgot Password" on login)
- **Two-factor auth (2FA)**: Settings → Security → Enable 2FA (authenticator app)
- **Billing**: Settings → Billing → View/change plan, update payment method
- **Export data**: Settings → Data → Export all workspace data (JSON/CSV)
- **Delete account**: Settings → Account → Delete (requires admin confirmation, 30-day grace period)

## Troubleshooting

### Common Issues
1. **Can't log in**: Clear browser cache, try incognito mode, check email for account status
2. **Slow loading**: Check internet connection, try different browser, clear cache
3. **Notifications not working**: Settings → Notifications → verify email/push enabled
4. **File upload failing**: Check file size (<100MB), check format, try different browser
5. **Integration not syncing**: Disconnect and reconnect the integration, check API key
6. **Mobile app crashing**: Update to latest version, clear app cache, reinstall
7. **Can't invite members**: Check your plan limits (Starter: 5 users, Professional: 50, Enterprise: unlimited)
8. **Task not saving**: Check internet connection, refresh page, try again
9. **Video call quality**: Use Chrome/Edge, check bandwidth, close other video apps
10. **Export not working**: Large exports take up to 10 minutes — check email for download link

### Known Limitations
- Maximum 1,000 tasks per project (contact support for increase)
- File upload limit: 100MB per file, 10GB total per workspace (Starter), 100GB (Professional), 1TB (Enterprise)
- API rate limit: 100 requests/minute (Professional), 1,000 requests/minute (Enterprise)
- Video calls: maximum 25 participants
- Custom fields: available on Professional and Enterprise plans only
