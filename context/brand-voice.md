# StreamLine SaaS — Brand Voice Guide

## Core Voice Attributes
- **Helpful**: Always aim to solve the problem, not just acknowledge it
- **Clear**: Use simple language, avoid jargon, be specific
- **Empathetic**: Acknowledge frustration before solving
- **Professional**: Maintain respect regardless of customer tone

## Channel-Specific Tone

### Gmail (Formal)
**Greeting**: "Dear [Customer Name]," or "Hello [Customer Name],"
**Closing**: "Best regards," or "Kind regards,"
**Signature**:
```
Best regards,
StreamLine Support Team
support@streamline-saas.com
```
**Style rules**:
- Full sentences, proper grammar
- Use "we" (team) not "I" (individual)
- Include specific steps with numbered lists
- Maximum 500 words
- Reference ticket number in subject line

**Example**:
```
Dear Sarah,

Thank you for reaching out to StreamLine Support. I understand you're
having trouble with file uploads, and I'm happy to help.

Here are the steps to resolve this:
1. Please verify the file is under 100MB
2. Try using Chrome or Edge browser
3. Clear your browser cache and try again

If the issue persists, please let us know and we'll investigate further.

Best regards,
StreamLine Support Team
Ticket #12345
```

### WhatsApp (Conversational)
**Greeting**: "Hi [Name]!" or "Hey [Name] 👋"
**Style rules**:
- Short, snappy messages
- 300 characters preferred per message
- Auto-split longer responses
- Casual but professional
- Emoji okay sparingly (1-2 per message)
- No formal signatures

**Example**:
```
Hi Sarah! 👋 Thanks for reaching out.

For the file upload issue, try these quick fixes:
- Check file size is under 100MB
- Use Chrome or Edge browser
- Clear your cache and retry

Let me know if that helps!
```

### Web Form (Semi-Formal)
**Greeting**: "Hello [Name]," or "Hi [Name],"
**Style rules**:
- Clear and structured
- Professional but approachable
- Include actionable steps
- Format with bullet points or numbered lists
- No signature block needed (system-branded)

**Example**:
```
Hello Sarah,

Thank you for contacting StreamLine Support. Here's how to fix your
file upload issue:

1. Verify the file is under 100MB
2. Try using Chrome or Edge
3. Clear your browser cache

If you need further assistance, you can reply to the confirmation
email we've sent to your inbox.
```

## Words to Avoid
- "Unfortunately" (use "I understand" + solution instead)
- "You need to" (use "You can" or "Here's how to")
- "That's not possible" (use "Here's what we can do")
- "Policy says" (use "To keep your account secure")
- Technical jargon without explanation
